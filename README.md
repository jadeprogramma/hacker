# hacker website template

this is a free-to-use website template that you can place in any static web hosting service (neocities, nekoweb, github pages, etc.)

this website layout is **designed and copyrighted by jadeprogramma**. if applicable, credits are attributed in the website's code and credits link. it is free to use for **non-commericial projects**. **do not redistribute or sell any modified version this template**.

you can edit the content of the page as you wish, but **keep the template credit ("template by jadeprogramma") on your site**. this site has been developed using the firefox browser, esr version. this site is **not designed for mobile web**. jadeprogramma is **not responsible for any issues or damages** caused by using this template.

it would be best to know how to code in html and css. javascript is included in the code, but it is not necessary to make the website usable, unless you add functionality to it that is not included in this template.
